# Images pour Vy'a Turismo

Placez vos images dans ce dossier avec les noms suivants :

## Images principales
- `hero.jpg` - Image du hero (étudiants en voyage) - Recommandé: 800x800px
- `ruins.jpg` - Ruines Jesuítiques - 400x600px
- `forest.jpg` - Nature du Paraguay - 400x600px

## Images des destinations (600x400px recommandé)
- `bertoni.jpg` - Monumento Moisés Bertoni
- `mbaracayu.jpg` - Reserva Mbaracayú
- `jesuitas.jpg` - La Tierra de los Jesuitas
- `cordillera.jpg` - La Cordillera
- `chaco.jpg` - El Chaco
- `neembucu.jpg` - Ñeembucú

## Formats supportés
- JPG, PNG, WebP
- Optimisez vos images avant de les télécharger (max 500KB par image)

## Comment utiliser
Une fois les images ajoutées ici, elles seront automatiquement disponibles dans le site à l'URL :
`/images/nom-de-fichier.jpg`
